Contextual Interactive Evaluation of TTS Models in Dialogue Systems

Anonymous

Submitted to Interspeech 2024

In this paper we evaluate our conversational TTS model with prosody control in a contextual interactive setting.
In order to do this we developed a "20 question" game using GPT-4 as a back-end, Google ASR and a set of evaluated TTS models.
We also develoved an interaction manager that kept track of the progress in the game and controlled the prosodic realization accordingly.

Using a "20 questions" animal guessing game we performed an evalaution of a Tacotron 2 TTS voice with prosody control (TTS-Pros),
controlling speaking style (from read to conversational) and prosody (mean pitch and speaking rate)
We compared it with a Tacotron voice without prosody control (TTS-T2) and a Matcha TTS voice (TTS-Matcha).
In the study 21 subjects were invited to our smart home lab to play the animal guessing game.
In the game the users and system took turn thinking about an animal or asking questions and guessing.
Each user played 4 games with the same system equipped with one of the three evaluated TTS systems.
Below are some examples of these interactions. For data privacy protecition, only ASR transcript of the user input is included.
